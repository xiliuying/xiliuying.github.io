云计算技术课程实验报告三

云计算技术课程实验报告三

基本信息

学号： 202100130126
姓名： 甘博文
班级： 计科21.1班
实验题目： 利用主流云平台搭建个人博客或网站
实验学时： 2
实验日期： 2024/3/22

实验目的

  1. 熟悉主流云平台的使用方法
  2. 熟悉主流云平台搭建个人博客或网站的方法
  3. 熟悉主流云平台搭建个人博客或网站的注意事项
  4. 熟悉主流云平台搭建个人博客或网站的常见问题及解决方法

硬件环境

联网的计算机一台

软件环境

Windows

实验步骤与内容

  1. 依次安装并测试git、nodejs和hexo
  2. git配置SSH key
  3. 在本地新建一个存放个人博客的文件夹
  4. 在github上新建仓库，修改博客配置文件中的_config.yml文件，修改部署文件中的deploy文件，部署到github仓库
  5. 利用git clone命令克隆fluid仓库到本地，修改部分属性以满足个人使用需求
  6. 撰写实验报告，利用hexo g、hexo s、hexo d命令测试本地页面并上传到github页面

结论分析与体会

本实验学习了如何在云端部署一个自己的网站。我一开始使用了阿里云，购买域名、服务器并安装了web服务器apache。但是不会的东西太多了，于是转用github+hexo配置个人博客，简单多了。
